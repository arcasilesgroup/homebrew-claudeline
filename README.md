# homebrew-claudeline

Homebrew tap for [`arcasilesgroup/claudeline`](https://github.com/arcasilesgroup/claudeline) — a cross-platform statusline for Claude Code.

## Install

```bash
brew tap arcasilesgroup/claudeline
brew install claudeline
```

## Updating

`bump.sh <version>` is used by maintainers to update the formula after a release.
